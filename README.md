# 🔐 Animated Login Interface (Login com Animação)

> Um portal de autenticação sofisticado e responsivo que prioriza a experiência do usuário (UX) através de uma interface limpa e opções flexíveis de acesso.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir uma **Página de Login** que não apenas cumpra sua função técnica, mas que também ofereça uma navegação visualmente agradável. 

A interface foca na clareza das informações e na facilidade de interação, incluindo campos para credenciais tradicionais e botões dedicados para autenticação via redes sociais.

## ✨ Destaques e Diferenciais Técnicos

* **Validação Nativa de Formulário:** Implementação do atributo `required` nos campos de entrada, garantindo a integridade dos dados antes da submissão.
* **Segurança na Interface:** Uso do tipo de entrada `password` para mascarar nativamente informações sensíveis do usuário.
* **Micro-interações Visuais:** Estruturação de elementos `.underline` sob os campos de entrada, preparada para animações de foco e feedback visual.
* **Integração de Login Social:** Design de botões para Twitter e Facebook integrados com ícones da biblioteca **Font Awesome (v5.15.2)**, demonstrando versatilidade em fluxos de acesso.
* **Arquitetura de Layout Modular:** Organização do código em containers lógicos (`.input-box`, `.option`, `.twitter`, `.facebook`), facilitando a manutenção e a escalabilidade do CSS.
* **Organização Profissional:** Separação clara entre a marcação estrutural (`index.html`) e a identidade visual (`css/style.css`), seguindo as melhores práticas de engenharia de software.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação e lógica de formulário.
* **CSS3:** Gerenciamento de layout, tipografia e design de componentes.
* **Font Awesome:** Iconografia profissional para suporte visual e social media.
